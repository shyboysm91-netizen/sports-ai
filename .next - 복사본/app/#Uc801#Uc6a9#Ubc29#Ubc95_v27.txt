Sports AI 전문가 리포트 v27

적용 방법
1. sports-ai 프로젝트 최상위에서 압축 해제
2. app 폴더 전체 덮어쓰기
3. 실행 중인 개발 서버 종료
4. npm run dev
5. 브라우저에서 Ctrl + F5
