Sports AI - 베트맨 자동 배당 연동 버전

변경 내용
1. 경기 상세 화면의 베트맨 수동 입력창 제거
2. /api/betman 자동 배당 조회 API 추가
3. 경기 날짜·원정팀·홈팀을 기준으로 베트맨 경기 자동 매칭
4. 승패 배당, 시장 환산 승률, AI Value 차이 자동 계산
5. 언더오버 기준점 수신 시 AI 예상 총점과 자동 비교
6. 배당 미수신 시 가짜 숫자나 고정 8.5를 사용하지 않고 '미수신' 표시
7. 베트맨 HTML/내장 JSON 구조 변경에 대비한 다중 파싱 로직 추가
8. TypeScript 검사 및 Next.js 프로덕션 빌드 통과

주의
