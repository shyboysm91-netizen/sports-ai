Sports AI 전문가 리포트 v28 오류 수정

오류 원인
- 실제 KBO 상태 변수는 awayStarter, homeStarter인데
  전문가 리포트 호출부에서 존재하지 않는
  awayStarterData, homeStarterData를 사용했습니다.

수정
- awayStarterData → awayStarter
- homeStarterData → homeStarter

적용 방법
1. sports-ai 프로젝트 최상위에서 압축 해제
2. app 폴더 전체 덮어쓰기
3. 실행 중인 개발 서버 종료
4. .next 폴더 삭제
5. npm run dev
6. 브라우저에서 Ctrl + F5
